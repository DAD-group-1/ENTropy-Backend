# ENTropy-Backend

Regroup all the microservices in a single repo for convenience

# Common packages used in multiple microservices

- [Swagger](https://www.npmjs.com/package/@nestjs/swagger): For API documentation and testing
- [AsyncAPI](https://github.com/flamewow/nestjs-asyncapi): For message-driven architecture documentation
- [TypeORM](https://typeorm.io/): For database interactions
- [NestJS Config](https://docs.nestjs.com/techniques/configuration): For managing environment variables and
  configuration
